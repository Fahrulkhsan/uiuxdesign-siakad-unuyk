# uiuxdesign-siakad-unuyk
Proyek ini merupakan hasil studi kasus untuk mata kuliah UI/UX Design di Universitas Alma Ata. Dalam proyek ini, saya melakukan analisis dan perancangan ulang (redesign) terhadap antarmuka (UI) dan pengalaman pengguna (UX) pada sistem informasi akademik yang digunakan oleh Universitas Nahdlatul Ulama Yogyakarta.
